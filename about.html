What about this project

This chrome application allow to manage a fence for table saw (or router table)

It's a fork of project available on github (https://github.com/sir-buckyball/chrome-gcode-sender).

This software is released under MIT licence

What libraries did you use?

  • AngularJS - MVW framework
  • Bootstrap - layout library (makes things pretty)
  • jQuery - general javascript utilities
  • chrome.serial - chrome serial API
  • paper.js - canvas rendering library
  • Moment.js - time library
  • Mousetrap - keyboard bindings
  • AngularHotkeys - bridge hotkeys and angular
